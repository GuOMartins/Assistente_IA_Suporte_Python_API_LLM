# Assistente_IA_Suporte_Python_API_LLM
Aplicação de IA em Python com integração a LLM via API. Desenvolve um assistente interativo para responder dúvidas sobre Python. Inclui Streamlit, backend em Python, controle de fluxo, tratamento de erros e engenharia de prompt, simulando ferramentas modernas de suporte inteligente.
